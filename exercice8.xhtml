Phase 2 : définir des requêtes xPath


  • Rechercher les uttérances (associées aux élements <u>) de l’opérateur :
    • //u[contains(@who,'O')]

  • Rechercher les uttérances (associées aux élements <u>) du client :
    • //u[contains(@who,'C')]

  • Rechercher les uttérances (associées aux élements <u>) qui contiennent le mot "quand" :
    • //u[contains(text (),'quand')]

  • Rechercher les uttérances (associées aux élements <u>) qui contiennent le caractère '?' :
    • //u[contains(text (),'?')]

  • Rechercher les uttérances (associées aux élements <u>) qui précèdent celles qui contiennent le caractère '?' :
    • //u[preceding-sibling::*[1][contains(text (),'?')]
