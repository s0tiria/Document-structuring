Phase 2 : définir des requêtes xPath


  • Rechercher tous les NOM :
    • //taggertoken[contains(@pos,'NOM')]/@wordform

  • Rechercher tous les VERBE :
    • //taggertoken[contains(@pos,'VER')]/@wordform

  • Rechercher tous les NOM précédés d’un DET :
    • //taggertoken[contains(@pos,'NOM')][preceding-sibling::*[1][contains(@pos,'DET')]]

  • Rechercher toutes les séquences NOM ADJ ou ADJ NOM :
    • //taggertoken[contains(@pos,'NOM')][following-sibling::*[1][contains(@pos,'ADJ')]] | //taggertoken[contains(@pos,'ADJ')][preceding-sibling::*[1][contains(@pos,'NOM')]]/@wordform
